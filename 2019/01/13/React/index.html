什么是 React

  • MVVM: (model - view - viewModel)
  • model: 指的是数据结构或者数据模型
  • view: 是展示数据的视图
  • viewModel: 是 model 和 view 的桥梁,使 view 发生变化的同时, model 也跟着变,反之亦然(也可以理解为viewModel的作用是处理 model 和 view 之间的数据逻辑)
