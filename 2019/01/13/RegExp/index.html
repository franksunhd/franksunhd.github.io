元字符

\d/\D

  • 匹配数字

\w/\W

  • 匹配单词字符

\s/\S

  • 匹配空白字符

\b/\B

  • 匹配单词边界

.

  • 匹配单个字符,除了换行符和行结束符

量词

n{X}

  • 匹配包含 X 个 n 的序列的字符串.

n{X,Y}

  • 匹配至少 X个,至多 Y个

n?

  • 匹配任何包含零个或一个 n 的字符串

?=n

  • 匹配任何其后紧接指定字符串 n 的字符串。

?!n

  • 匹配任何其后没有紧接指定字符串 n 的字符串。
