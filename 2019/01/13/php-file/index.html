PHP 文件操作/目录操作/追加内容

发表于 2019-01-13 | 分类于 PHP | 字数统计 1.8k 字 | 阅读时长 7 分钟
php file

文件操作

打开文件/读取文件/关闭文件

从资源中读取一行

  • fgets(‘句柄资源’,’长度’)
  • 从资源中读取一行,如果碰到换行符, EOF, 或者读到文件的最后,会停止读取

把文件按照数组的方式读取出来

  • file(文件路径)
  • 它使用的是换行符隔开元素

把文件按照字符串的方式打开

  • file_get_contents(文件路径)

复制文件

  • copy(老文件,新文件)

重命名文件

  • rename(老文件,新文件)

删除文件

  • unlink(文件路径)

获取文件的最终修改时间

  • filemtime(文件路径)

判断文件是否存在

  • file_exists(文件名)
  • 返回值:存在返回 true 不存在返回 false

判断文件是否可读

  • is_readable(文件路径)
  • 返回值:可读返回 true 不可读返回 false

判断文件是否可写

  • is_writeable(文件路径)
  • 返回值:可写返回 true 不可写返回 false

目录操作

打开目录/读取目录元素/关闭目录

返回的是路径子目录组成的数组

  • scandir(路径)

删除目录

  • rmdir(文件路径)
  • 删除文件夹该文件夹必须为空

创建目录 mkdir

  • (创建的目录名.mode 权限设置(默认是0777))

basename(‘文件路径’,’扩展名’)

pathinfo(文件路径, options(返回的信息))

判断是否有这个目录

  • file_exists(目录名)
  • 返回值:有返回 true, 没有返回 fasle

追加内容

写入文件

  • fwrite(资源句柄,要添加的内容)
  • 出错报 false
  • 返回添加内容的长度

file_put_contents

  • file_put_contents(文件名,要添加的资源, FILE_APPEND);
  • 返回的追加的字符数
  • 注意:最后一个参数没有,则为先清空,而后重新写入
