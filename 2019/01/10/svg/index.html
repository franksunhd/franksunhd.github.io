什么是 SVG?

  • SVG 是可伸缩的矢量图形,通过 XML方式来绘制图形,并且放大或者缩小,或者改变尺寸都不会影响图片的质量
  • SVG 是 万维网联盟的标准.

SVG 和 Canvas 的区别

  • SVG 是基于 XML 绘制的2d 图形 , canvas 是通过 JavaScript 绘制的2d 图形
  • canvas 可以以 .png 和 .jpg 格式进行图片保存 , 而 SVG 格式的图片不可以保存
