jQuery中的事件操作

发表于 2019-01-14
jQuery

鼠标事件

click()

  • 鼠标单击事件
  • 为 JavaScript 的”click” 事件绑定一个处理器，或者触发元素上的 “click” 事件。
  • click是添加单击事件的方法
  • 匿名函数就是没有名字

dblclick()

  • 鼠标双击事件
  • 为JavaScript 的 “dblclick” 事件绑定一个处理函数，或者触发元素上的 “dblclick” 事件。
  • 双击时会先触发单击！！！！！

contextmenu()

  • 鼠标右击事件
  • 为 JavaScript 的”contextmenu” 事件绑定一个处理器，或者触发元素上的 “contextmenu” 事件。

mouseover()

  • 鼠标悬停事件

mouseout()

  • 鼠标离开事件

resize()

  • 浏览器窗口大小改变时触发

键盘事件

keydown()

  • 按下事件
  • keysdown()识别所有按键

keypress()

  • 按下事件
  • keypress() 识别字母数字键

keyup()

  • 抬起事件

事件绑定

on()方法

  • 在选定的元素上绑定一个或多个事件处理函数。
  • 使用on方法进行事件绑定,通用的事件绑定方法
  • on方法可以自定义事件
  • on方法也可以绑定多个事件处理函数

one()

  • 只响应一次

trigger()方法

  • 触发自定义的事件
  • trigger触发自定义的事件
  • 根据绑定到匹配元素的给定的事件类型执行所有的处理程序和行为。

off() 解除事件绑定

  • 移除一个事件处理函数。

事件切换

hover()

  • 将二个事件函数绑定到匹配元素上，分别当鼠标指针进入和离开元素时被执行。

表单事件

.focus() 获得焦点

.blur() 失去焦点

.change()

事件对象

e.pageX

  • 鼠标相对于文档的左边缘的位置（左边）。

e.pageY

  • 鼠标相对于文档的顶部边缘的位置（顶部）

e.preventDafault()

  • 如果调用这个方法，默认事件行为将不再触发。

e.stopPropagetion()

  • 不触发任何前辈元素上的事件处理函数。

e.which()

  • 针对键盘和鼠标事件，这个属性能确定你到底按的是哪个键。
