C语言结构体有哪些?

  • 构造数据类型是根据已定义的一个或多个数据类型用构造的方法来定义的.

C语言中构造类型有哪些？

  • 数组类型
  • 结构体类型
  • 共用体(联合)类型

结构体的内存分析

  • 结构名只能表示一个结构形式,编译系统并不会对它分配内存空间。只有当某变量被说明为这种类型的结构时,才对该变量分配存储空间。
  • 结构体占用的内存空间是每个成员占用的字节数之和

结构体对齐

  • （对齐模数 - k）
  • 结构体对齐有两个方面的含义：
  • 结构体总长度
  • 结构体内各数据成员的内存对齐，即该数据成员相对结构体起始位置
  • 结构体的长度应该是模数的整数倍
  • pragma pack 可以指定内存的对齐长度

结构类型定义作用域

概述

  • 结构类型定义在函数内部的定义域域局部变量的作用域是相同的
  • 函数外部定义的结构体类型类似全局变量
  • 作用域: 从定义的那一行开始直到本文件结束为止

分类

  • 根据作用域可分为：
  • 全局结构体 (到文件末尾结束)
  • 局部结构体 (到函数{}结束)

结构体数组

概念

  • 结构数组的每一个元素都是具有相同结构类型的下标结构变量

结构体指针

指向结构体变量的指针

  • 一个指针变量当用来指向一个结构变量时，称之为结构指针变量

  • 结构指针变量中的值是所指向结构变量的首地址,通过结构指针即可访问该结构变量

  • struct 结构名 *结构指针变量名

  • 结构指针变量必须先赋值后使用

  • 赋值是把结构变量的首地址赋予该指针变量,不能把结构名赋予该指针变量

  • 结构名只能表示一个结构形式

  • 结构指针变量,就是用来存储结构体变量地址的，就能更方便地访问结构变量的各个成员

结构体嵌套

  • 成员也可以又是一个结构,即构成了嵌套的结构。
  • 嵌套定义自己的指针

结构体和函数

  • 结构体成员属性作为函数的参数 (值传递)
  • 结构体名作为函数的参数
  • 结构体指针作为函数的参数

枚举

  • 列举所有可能的取值
  • 被说明为该”枚举”类型的变量取值不能超过定义的范围
  • 枚举类型是一种基本数据类型,而不是一种构造类型,因为它不能再分解为任何基本类型。
